QueueManager.php : 33

Main QueueManager.

  • Build bus from provided handlers (simple auto-register provided handlers).
  • Manage drivers.
  • Dispatch tasks (store to driver).
  • runWorker() polls driver and dispatches to bus; handles retry with exponential backoff.
